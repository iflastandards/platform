# UNIMARC Data Visualization

Interactive visualizations of UNIMARC data relationships and structures.

<!-- Visualization component will be added when available -->

## Available Visualizations

- **Element Set Overview**: Visual representation of all element sets
- **Vocabulary Relationships**: Network graph of vocabulary connections
- **Field Usage Statistics**: Charts showing field frequency and usage
- **Language Coverage**: Geographic visualization of language support
- **Historical Evolution**: Timeline of UNIMARC development

## Features

- Interactive exploration
- Export visualization as image
- Filter by category or type
- Zoom and pan controls
