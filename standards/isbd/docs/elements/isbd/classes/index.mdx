# Classes

This section defines the fundamental classes used in ISBD resource description.

## Classes in ISBD

### Resource (C2001)
The core class representing an entity, tangible or intangible, that comprises intellectual and/or artistic content and is conceived, produced and/or issued as a unit, forming the basis of a single bibliographic description.

<InLink href="/docs/elements/isbd/classes/C2001">View Resource class details</InLink>


In the context of ISBD, the core concept of 'Resource' is fundamental, forming the very basis of bibliographic description. The ISBD framework specifies the requirements for describing and identifying these resources to ensure global consistency and facilitate the international exchange of bibliographic records.
