# Place Of Publication, Production, Distribution Statement Encoding Scheme

## Element Reference
<ElementReference frontMatter={frontMatter} />

## Definition

<div className="guid">
This refers to element <strong>4.1</strong> within the <strong>Publication, Production, Distribution, Etc. Area (Area 4)</strong>. It specifies the <strong>name of the place associated with the publisher, producer, or distributor</strong>, or the place from which the resource was issued or distributed.
</div>

## Required Elements

<div className="guid">
<ul>
  <li><strong>Place name (Mandatory):</strong> The name of the city or town (e.g., "London," "Baltimore"). If unknown or uncertain, it can be supplied in square brackets, possibly with a question mark.</li>
  <li><strong>Larger place (Optional):</strong> The name of a country, state, etc., added for identification (e.g., "Washington, D.C.", "Cambridge, Mass."). This is preceded by a comma if from the source, or in square brackets if supplied.</li>
  <li><strong>Full address (Optional):</strong> The full address of the publisher, producer, or distributor, if necessary for identification. Enclosed in parentheses if from source, or square brackets if supplied.</li>
  <li><strong>Alternative/corrected version (Optional):</strong> If the given place name is incorrect, a correction can be supplied (e.g., "Paris [i.e. Leiden]", "Christiania [Oslo]").</li>
</ul>
</div>

## Punctuation Rules

<div className="guid">
<ul>
  <li>A <strong>second or subsequent place</strong> is preceded by <strong>;</strong> (unless a linking word or phrase is given).</li>
  <li><strong>Supplied information</strong> (e.g., unknown place, additions for identification) is enclosed in <strong>square brackets [ ]</strong>.</li>
  <li>The <strong>abbreviation s.l. (sine loco)</strong> is supplied in square brackets if no place can be determined.</li>
</ul>
</div>

## Relationship to ISBD Standards

<div className="guid">
This is element <strong>4.1</strong> within Area 4. It is a mandatory element in the bibliographic description.
</div>

## Relationship to Other SES

<div className="guid">
<ul>
  <li>It is a component of the "Publication, Production, Distribution, Etc Area Encoding Scheme."</li>
  <li>It is inherently linked to the "Name Of Publisher, Producer, Distributor Statement Encoding Scheme" (element 4.2), as the place is associated with that name.</li>
  <li>Changes in the place of publication for continuing resources or multipart monographic resources are noted in the <strong>Note Area (Area 7)</strong>.</li>
</ul>
</div>
