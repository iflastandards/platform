# FRBRer model

This is the element set of native RDF classes and properties described in the current text (Feb 2009) of the Functional Requirements for Bibliographic Records (FRBR) entity-relationship model.

This element set contains 216 elements.

## Overview

The FRBRer model element set provides a structured way to describe frbr - functional requirements for bibliographic records resources.
