# Attribute Elements

Attribute elements describe the physical characteristics and content attributes of resources.

## Categories

### Physical Attributes
- **has extent** - Number and type of units
- **has dimensions** - Measurements of the resource
- **has material** - Physical material composition

### Content Attributes
- **has content form** - Fundamental form of content expression
- **has media type** - Type of carrier and intermediation device
- **has content qualification** - Additional content characteristics

### Technical Attributes
- **has digital file characteristics** - Properties of digital files
- **has sound characteristics** - Audio properties
- **has video characteristics** - Visual motion properties

## Attribute Characteristics

Attribute elements typically:
- Have domain: Resource
- Have range: Literal or controlled vocabulary terms
- Describe objective characteristics
- Support technical metadata needs

*Note: This page lists representative examples. The full set of attribute elements will be populated through the import process.*
