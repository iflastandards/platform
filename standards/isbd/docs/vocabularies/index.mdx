# ISBD Vocabularies

The ISBD namespace includes seven concept vocabularies that provide controlled terminologies for content forms, qualifications, and media types.

## Overview

These vocabularies support ISBD Area 0 (Content form and media type area) by providing standardized terms for describing:
- The fundamental forms in which content is expressed
- Qualifications that further specify content characteristics
- The types of carriers used to convey content

## Technical Details

### Namespace Pattern
- Base URI: `http://iflastandards.info/ns/isbd/terms/`
- Vocabulary pattern: `{base}{vocabulary}/{term}`
- Example: `http://iflastandards.info/ns/isbd/terms/contentform/T1001`

### Term Identifiers
All vocabulary terms use the T-prefix numbering system:
- T1001-T1999: Reserved for vocabulary terms
- Each vocabulary has its own number range

## Using ISBD Vocabularies

These vocabularies are designed to be used with ISBD elements, particularly:
- `has content form` (P1022)
- `has media type` (P1023)
- `has content qualification` (various sub-properties)

Example:
```turtle
@prefix isbd: <http://iflastandards.info/ns/isbd/elements/> .
@prefix isbdcf: <http://iflastandards.info/ns/isbd/terms/contentform/> .

<http://example.org/resource/1>
    isbd:P1022 isbdcf:T1003 . # has content form: text
```
