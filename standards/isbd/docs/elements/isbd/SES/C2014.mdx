# Numbering Area Encoding Scheme

## Element Reference
<ElementReference frontMatter={frontMatter} />

## Definition

<div className="guid">
This is a sub-area within the <strong>Material or Type of Resource Specific Area (Area 3)</strong>, specifically for <strong>serials</strong>. It records the <strong>numeric and/or chronological designation</strong> of the first and/or last issue or part of the serial.
</div>

## Required Elements

<div className="guid">
<ul>
  <li><strong>Numeric designation (Mandatory if available):</strong> Numbers (arabic numerals preferred) with or without accompanying characters/words (e.g., "Vol. 1, no. 1", "Bd. 1–").</li>
  <li><strong>Chronological designation (Mandatory if available):</strong> Dates (e.g., "Jan. 1970", "1925–").</li>
  <li><strong>Parallel numbering system (Optional):</strong> Designations in another language or script for the same issues.</li>
  <li><strong>Alternative numbering system (Optional):</strong> A different numbering system used concurrently.</li>
  <li><strong>New sequence of numbering (Optional):</strong> The designation and numbering for a new sequence when the serial continues under a new numbering scheme (e.g., "new ser., v. 1–").</li>
</ul>
</div>

## Punctuation Rules

<div className="guid">
<ul>
  <li>A <strong>hyphen (–)</strong> after the number/date of the first issue links to the last issue or indicates it is continuing.</li>
  <li>If both <strong>numeric and chronological designations</strong> are present, the date is enclosed in <strong>parentheses ( )</strong> after the number, unless the number is a division of the date.</li>
  <li>The <strong>second and each subsequent system of numbering</strong> (parallel or alternative) is preceded by <strong>=</strong>.</li>
  <li>A <strong>new sequence of numbering</strong> is preceded by <strong>;</strong>.</li>
</ul>
</div>

## Relationship to ISBD Standards

<div className="guid">
This is element <strong>3.3</strong> within Area 3. It provides specific, standardized numbering information critical for identifying and managing serial resources.
</div>

## Relationship to Other SES

<div className="guid">
<ul>
  <li>It is a component of the "Material Or Type Of Resource Specific Area Encoding Scheme."</li>
  <li>The dates in this area may not coincide with publication dates in the <strong>Publication, Production, Distribution, Etc. Area (Area 4)</strong>.</li>
  <li>Major changes in numbering can require a new description for the serial (see Area A.2.6).</li>
  <li>Notes on complex, irregular numbering, or explanations for omissions are given in the <strong>Note Area (Area 7)</strong>.</li>
</ul>
</div>
