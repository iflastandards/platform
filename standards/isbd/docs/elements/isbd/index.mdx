## Element Set Details

- **Namespace URI**: `http://iflastandards.info/ns/isbd/elements/`
- **Preferred Prefix**: `isbd:`
- **Element ID Pattern**: P + number (e.g., P1001, P1002)

## RDF Constraints

All elements in this set have defined domains and ranges:

- **Domains**: Typically `Resource` or specific entity types
- **Ranges**: `Literal` for textual values, or specific RDF classes for relationships

These constraints ensure semantic consistency when using ISBD elements in linked data applications.
