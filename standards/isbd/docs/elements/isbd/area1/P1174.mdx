## Additional Information

- **Scope and Purpose:** The element provides the title of the series or multipart monographic resource in another language or script, aiding in identification, especially in multilingual contexts. It enhances the discoverability of resources for users searching in different languages.

- **User Tasks Supported:**
    - Resource discovery in multilingual environments
    - Identification of series across language barriers
    - Facilitating interoperability of bibliographic data across languages


## Element Values

- **Rules/Guidelines:** The parallel title should be transcribed exactly as to wording from the resource, but not necessarily as to capitalization and punctuation. If a statement of responsibility can be linguistically linked to the parallel title, it is transcribed as part of the parallel title. (1.2.5.1, adapted for Area 6)

- **Value Constraints/Formats:** Should be a valid title string in the target language/script.

- **String Encoding:**  Follows the general guidelines for language and script of the description (A.5).
