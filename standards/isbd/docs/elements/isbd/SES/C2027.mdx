# Terms Of Availability Statement Encoding Scheme

## Element Reference
<ElementReference frontMatter={frontMatter} />

## Definition

<div className="guid">
This refers to element <strong>8.3</strong> within the <strong>Resource Identifier and Terms of Availability Area (Area 8)</strong>. It provides a statement of the <strong>conditions under which a resource may be obtained</strong>, typically its price, or a brief statement if it's not for sale (e.g., "for hire," "free to educational institutions").
</div>

## Required Elements

<div className="guid">
The element is <strong>Optional</strong>.
<ul>
  <li><strong>Price (Optional):</strong> Given with the international standard code for the currency (e.g., "GBP 2.05," "EUR 950").</li>
  <li><strong>Other terms (Optional):</strong> A brief statement for non-sale availability (e.g., "not for sale," "for hire," "free loan").</li>
  <li><strong>Qualifications (Optional):</strong> Brief statements (in parentheses) clarifying the terms (e.g., "(to members)", "(until 1 January 1977)", "(annual subscription)").</li>
</ul>
</div>

## Punctuation Rules

<div className="guid">
<ul>
  <li>The <strong>terms of availability</strong> are preceded by <strong>:</strong>.</li>
  <li><strong>Qualifications</strong> are enclosed in <strong>parentheses ( )</strong>.</li>
  <li>If multiple terms or qualifications are needed, they are separated as appropriate, usually by <strong>:</strong> for distinct terms or <strong>;</strong> for multiple qualifications.</li>
</ul>
</div>

## Relationship to ISBD Standards

<div className="guid">
This is element <strong>8.3</strong> within Area 8. It provides practical information about access to the resource.
</div>

## Relationship to Other SES

<div className="guid">

</ul>
</div>
