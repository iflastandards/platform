# UNIMARC Export Tools

Export UNIMARC data in various formats.

<!-- ExportTools component will be added when available -->

## Export Formats

- **RDF/XML**: Resource Description Framework XML format
- **JSON-LD**: JSON for Linked Data
- **Turtle**: Terse RDF Triple Language
- **CSV**: Comma-separated values for spreadsheet applications
- **MARC21**: Convert to MARC21 format
- **Dublin Core**: Map to Dublin Core metadata

## Features

- Batch export multiple records
- Custom field mapping
- Format validation
- Download or API integration
