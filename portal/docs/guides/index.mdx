## üí° Tips for Success

### Choose the Right Tools
- **Small vocabularies (< 20 terms)**: Use the text/YAML method
- **Large vocabularies or team collaboration**: Use the CSV/spreadsheet method
- **Complex relationships**: Consider using the profiles system

### Follow Best Practices
- Write clear, concise definitions
- Use scope notes to clarify usage
- Provide examples when helpful
- Maintain consistency across your vocabulary

### Get Help When Needed
- Review the component documentation for usage details
- Check the examples for patterns to follow
- Ask the technical team for assistance with complex scenarios
