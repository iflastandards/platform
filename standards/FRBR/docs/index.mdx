## Overview

This page introduces the FRBR Element Sets and Value Vocabularies.

## FRBR Element Sets

* **FRAD model** - This is the element set of native RDF classes and properties described in the Functional Requirements for Authority Data model. The model also uses elements previously described in the Functional Requirements for Bibliographic Records (FRBR) entity-relationship model.

* **FRBRer model** - This is the element set of native RDF classes and properties described in the current text (Feb 2009) of the Functional Requirements for Bibliographic Records (FRBR) entity-relationship model.

* **FRBRoo model** - Element set of the Functional Requirements for Bibliographic Records object-oriented extension to the the CIDOC Conceptual Reference Model (CRM)

* **FRSAD model** - This is the element set of native RDF classes and properties described in the Functional Requirements for Subject Authority Data model. The model also uses elements previously described in the Functional Requirements for Bibliographic Records (FRBR) entity-relationship model.

## FRBR Value Vocabularies

* **FRBRer User Tasks** - Based on the current text, Feb 2009.

* **FRSAD User Tasks** - User tasks for Subject Authority Data.
